useForm 钩子 ​

  • 用于生成表单相关事件方法

使用 ​

js
import { useForm } from "profield";

const { formref, formModel } = useForm();

参数 ​

参数说明类型默认值必填
emit表单提交事件emit-否
props表单属性props-否
columns表单字段columnItem[]-否

返回值 ​

参数说明类型
formref表单refRef<ElForm>
formModel表单数据Ref<Recordable>
submit表单提交方法(e: Event) => void
reset表单重置方法() => void
makeRule生成规则(rule: columnItem) => RuleObject[]
